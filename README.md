# FrameReady DEV Files

FrameReady is a multi-file Filemaker Pro solution that allows users to manage their Framing company all in one powerful tool.

## 🚀 Key Features
- **Binary Files**: These files are binary .fmp12 files which cannot be opened in a text editor, they must be opened only in Claris Filemaker Pro
- **Multi File**: The file 'FrameReady.fmp12' is the launcher file and all of the other files work together in a multi file environment
- **All In One**: The code that drives the UI of the program and interacts with the data is located inside of the files under the 'Script Workspace'
- **Database is Filemaker**: The database which holds all of the data and relationships is not MySQL or SQL but is inside the files themselves under File > Manage > Database

## 🛠️ Getting Started

### Prerequisites
Download all of the files and store them inside the same folder on your machine locally. Then, within that folder, click on the FrameReady.fmp12 launcher to open the files.
